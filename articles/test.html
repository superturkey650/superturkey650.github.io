Getting Started with This Blog

This is a static blog: the homepage lists posts, and each post is its own HTML page. Tailwind is loaded via CDN to keep things simple.

Folder structure

your-blog/
index.html
about.html
articles/
getting-started.html
responsive-design.html
tailwind-tips.html
notes-on-writing.html

Adding a new post

  1. Duplicate an existing article file.
  2. Update the title, date, and content.
  3. Add a new card on index.html linking to it.

Deploy

You can deploy as-is to GitHub Pages, Netlify, or any static host.
